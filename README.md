#### Explaining Image Classification - Ein Methodenvergleich

Problembeschreibung: Ein Convolutional Neural Network (CNN) ist ein Machine Learning-Verfahren
in der Bildklassifikation, das sehr hohe Genauigkeit erreicht. Daher werden sıe zunehmend ın
kritischen Bereichen wie beispielsweise der Diagnose von Lungenentzündungen verwendet. Die
Entscheidungen dieser komplexen Modelle sınd allerdings nicht mehr vom menschlichen Anwender
nachvollziehbar! Daher gibt es in der Forschung zur Explainable Al (XAl) Ansätze, welche
sogenannte Counterfactuals (CF) bestimmen. Diese stellen dar, welche Bildbereiche einer krankhaften
Lunge sich wie verändern müssten, um laut dem CNN zu z.B. einer gesunden Lunge beizutragen.
Aufgaben: - Praktische Implementierung und Evaluation von CF-Ansätzen aus bestehender Literatur
(in Python) - Strukturierte Aufbereitung und Diskussion der Ergebnisse Startliteratur: Moreira,
Catarina, et al. "Benchmarking instance-centric counterfactual algorithms for XAl: from white box to
black box." ACM Computing Surveys 57.6 (2025): 1-37.
